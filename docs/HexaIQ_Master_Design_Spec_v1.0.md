# HexaIQ Master Design Specification v1.0

## 1. 프로젝트 비전

HexaIQ는 기존 IQ 점수 중심 평가를 넘어 6가지 인지능력을 분석하고, AI 기반 학습 추천과 장기 성장 추적을 제공하는 플랫폼이다.

서비스 사이클:

```text
검사 → AI 분석 → 맞춤훈련 → 성장기록 → 재검사 → 성장예측
```

## 2. 핵심 인지영역

1. 수리논리
2. 공간지각
3. 논리추론
4. 언어유추
5. 작업기억
6. 추상패턴 인식

## 3. 대상

- 유치부
- 초1~2
- 초3~4
- 초5~6
- 중학생
- 고등학생
- 성인

연령과 수행능력에 따라 적응형 난이도를 적용한다.

## 4. 검사 종류

### 기본형

- 약 10분
- 영역당 5문항
- 총 30문항
- 무료
- 결과 확인 전 보상형 광고 2회

### 심화형

- 약 30분
- 영역당 20문항
- 총 120문항
- 각 영역 완료 시 보상형 광고 1회
- 총 6회 광고

### 전문 심사용

- 약 60분
- 영역당 40문항
- 총 240문항
- 적응형(CAT)
- 광고 없음
- USD 5 결제
- PDF 리포트 제공

## 5. 문제 생성 엔진

고정 문제은행이 아닌 템플릿 기반 생성 방식을 사용한다.

구조:

```text
문제유형 → 난이도(Level 1~10) → 연령 → 랜덤 Seed → 문제 생성
```

동일 문제 반복을 최소화한다.

## 6. 결과 리포트

- 육각형 능력 그래프
- 종합 인지지수(참고용)
- 영역별 점수
- 백분위
- 강점/약점
- AI 분석
- 학습전략
- 추천 문제집
- 추천 참고서
- 추천 훈련앱

## 7. 성장 시스템

- 검사 이력
- 육각형 변화
- 월별 성장
- AI 성장예측
- 경험치
- 레벨
- 업적
- 배지

## 8. 사용자 관리

기기당 최대 3개 프로필을 지원한다.

프로필 정보:

- 이름
- 학년
- 연령대
- 성별(선택)
- 아바타

각 프로필은 독립적인 성장 데이터를 가진다.

## 9. 훈련 생태계

HexaIQ는 검사 허브 역할을 한다.

연동 앱:

- Hexa Math
- Hexa Space
- Hexa Logic
- Hexa Language
- Hexa Memory
- Hexa Pattern

AI가 부족한 영역을 자동 추천한다.
