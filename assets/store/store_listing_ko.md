# HexaIQ Play Store Listing

## 앱 이름

HexaIQ

## 짧은 설명

도형, 패턴, 수리, 기억력 문제로 인지 능력을 확인하는 IQ 테스트 앱

## 긴 설명

HexaIQ는 수리 논리, 언어 이해, 공간 지각, 패턴 인식, 작업 기억, 처리 속도 등 다양한 인지 영역을 기반으로 사고력을 확인할 수 있는 IQ 테스트 앱입니다.

도형, 색상, 아이콘, 시각 패턴, 수열, 논리 문제를 통해 사용자의 문제 해결 방식과 영역별 강점을 분석합니다.

주요 기능

- 빠른 IQ 테스트
- 기본 IQ 테스트
- 심화 IQ 테스트
- 전문 IQ 테스트
- 영역별 점수 분석
- 육각형 그래프 리포트
- PDF 결과 리포트
- 한국어 결과 설명
- 다크모드 지원

검사 결과는 사용자의 사고 유형과 인지 영역별 특성을 이해하기 위한 참고 자료입니다. 의료적, 임상적, 교육기관 공식 진단을 대체하지 않습니다.

일부 기능은 광고 시청 또는 인앱결제를 포함할 수 있습니다.

## 콘텐츠 등급 메모

교육, 퍼즐, 인지 테스트, 참고용 검사 앱입니다. 결과는 공식 진단이 아닙니다.

## 광고 및 결제 고지

무료 검사에는 일부 광고가 포함될 수 있습니다. 전문 검사는 USD $4.90 일회성 인앱결제로 제공됩니다.
